===============
MIDAS converter
===============

This repository contains ROS2 packages and scripts for converting RGB images to depth images using the **MiDaS model**. It is designed for pre-processing datasets for drone navigation and AI projects.

# Prerequisites
Follow the step below to setup the environment:
```bash
python -m venv .venv
source .venv/bin/activate  # Linux/macOS
.venv\Scripts\activate     # Windows
pip install -r requirements.txt
